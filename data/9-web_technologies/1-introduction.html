The objective of this course is for you to develop an understanding of what the internet is, and how it works. At the end of this course you will be able to create interactive web pages and will be able to write simple programming code in JavaScript.

How to use these pages

This course is designed to run at your own pace. If you fly through it, great! Move on to the next page, but make sure you answer all questions and complete any extension tasks first. These will make sure you understand what you're doing.

If you find you're going slower, that's ok too. Ask for help if you need it.

But most importantly, READ everything on these pages. Don't just skip to the code sections, you'll miss a lot on the way.

Don't copy and paste the code either. By typing it, you're improving your typing skills, and learning how each line works.

As much as possible, technical terms will be in italics, while instructions or important notes will be in bold.

Blocks of code will appear in dark boxes.

References to small sections of code will be in boxes like this.

At then end of this lesson, you will have a basic understanding of how to save your work, how to upload files to Connect, and what the internet is.

Setup tasks

Before we begin, create a Year 9 Digital Technologies folder in your My Documents for this course. Make sure you are working under Computer -> My Documents, not Libraries -> Documents.

Inside the folder you have just created, add another new folder called HTML

Text Editors

Web pages are simple text files, so we will be using a text editor, VS Code. Find and open VS Code now.

Background Information

Create a new text file in VS Code. At the top of the page write your name. Underneath, write Background Information.

Save your page in the HTML folder you created as background_information.txt

Answer the following questions in full sentences and/or paragraphs in your text file:

  1. What is a computer?
  2. What is a network?
  3. What is a server?
  4. What is the internet?
  5. What is a browser?
  6. List all the browsers you have used on ALL your devices.
  7. Which browser do you prefer to use, and why?

When you have finished writing your answers in full sentences, save your file and upload it to Connect.
